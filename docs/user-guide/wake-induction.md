# 起床誘導

起床誘導は、起きる時刻に向けて明るさと色温度を戻し、復帰しやすい状態へ整える機能です。
左メニューの「睡眠サポート」から基本設定できます。

## 基本設定

1. 左メニューの「睡眠サポート」を開きます。
2. 「今夜の設定」で「起床サポート」を ON にします。
3. 「時刻の詳細」に表示される起床時刻を確認します。
4. 時刻が未設定の場合は、「スケジュール設定を開く」から起床予定時刻を入力します。
5. 最初は「標準」プリセットのまま使います。

## 起床時刻を設定する

起床誘導は、スケジュールの起床予定時刻にあわせて動きます。

1. 左メニューの「スケジュール」を開きます。
2. 固定時刻で使う場合は、「起床予定時刻」を入力します。
3. 日によって時刻が違う場合は、「スケジューラーを有効化」を ON にします。
4. カレンダー、テンプレート、または「まとめて設定」で起床予定を入力します。

## 起床までの遷移時間

「睡眠・起床の調整」では、起床予定時刻までに何分かけて起床誘導を進めるかを設定できます。

1. 左メニューの「睡眠・起床の調整」を開きます。
2. 「時間設定の調整」を確認します。
3. 「起床までの遷移時間」に分数を入力します。
4. 睡眠モードに入ってから一定時間後に起床誘導したい場合は、「睡眠開始からの起床サポート」を ON にします。

## 画面の戻し方を調整する

「睡眠・起床の調整」の「起床誘導中」では、起床時の明るさと色温度を調整できます。

1. 「明るさ」を確認します。
2. 色温度制御を使っている場合は、「色温度」を確認します。
3. 明るく戻りすぎる、または暗すぎる場合だけ調整します。

## 詳細な誘導・睡眠設定

「詳細な誘導・睡眠設定」では、起床誘導プロファイルの詳細を設定できます。

1. 「起床誘導プロファイル」を確認します。
2. 「明るさを変更する」を ON にすると、起床誘導時の明るさを指定できます。
3. 色温度制御を有効にしている場合は、「色温度 (K)」を指定できます。
4. 「手動実行時の遷移時間 (秒)」と「自動実行時の遷移時間 (秒)」を必要に応じて調整します。

!!! note "音量の扱い"
    起床誘導では個別の音量を指定せず、入眠開始前に記録した音量へ戻します。

!!! tip "慣れるまでは変更しすぎない"
    起床誘導は予定時刻と遷移時間の影響が大きいです。
    まずは予定時刻を正しく入れて、必要になったら明るさや色温度を調整してください。

## 困ったとき

起床誘導が始まらない場合は、「起床サポート」が ON か、起床予定時刻が設定されているかを確認してください。
起床時刻が日によって変わる場合は、[スケジュール](scheduler.md)の設定も確認してください。
