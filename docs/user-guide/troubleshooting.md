# トラブルシューティング

SleepVivo は現在 Pre-Alpha 版です。
問題が起きた場合は、Discord でハルジオン @fleabane_haru に連絡してください。
小さな表示崩れ、違和感、わかりにくい説明も報告してもらえると助かります。

## 連絡前に確認すること

1. SleepVivo を再起動します。
2. SteamVR を再起動します。
3. VR 機器が SteamVR から見えているか確認します。
4. 直前に変更した設定があれば、内容をメモします。
5. 可能であれば、問題が起きた画面のスクリーンショットを用意します。

## 起動しない

1. インストーラーが現在の配布元から受け取ったものか確認します。
2. Windows の警告やエラー文を確認します。
3. SteamVR を起動した状態でもう一度 SleepVivo を起動します。
4. それでも起動しない場合は、表示されたエラー文を添えて連絡します。

## 入眠誘導・起床誘導が動かない

1. 左メニューの「睡眠サポート」を開きます。
2. 「入眠サポート」または「起床サポート」が ON か確認します。
3. 「時刻の詳細」に時刻が表示されているか確認します。
4. 時刻が未設定の場合は、[スケジュール](scheduler.md)で入眠予定時刻と起床予定時刻を入力します。
5. 日ごとの予定を使っている場合は、「スケジューラーを有効化」が ON か確認します。

## スケジュールが反映されない

1. 「スケジュール」を開きます。
2. 固定時刻で使っているか、スケジューラーで使っているか確認します。
3. スケジューラーを使っている場合は、カレンダーの日付を選びます。
4. 「就寝予定（前日）」と「起床予定」が入っているか確認します。
5. 「まとめて設定」を使った場合は、対象期間と曜日を確認します。

## 通知や音量がおかしい

1. 「設定」から「通知の設定」を開きます。
2. 「通知先」を確認します。
3. 「一般通知の音量」を調整し、「テスト」で確認します。
4. 「睡眠・起床の調整」で「音量対象デバイス」が意図したデバイスになっているか確認します。

## 連携がうまくいかない

1. 「自動化・連携」または「設定」内の関連ページを開きます。
2. VRChat、Pulsoid、Home Assistant、Discord などの接続状態を確認します。
3. OSC を使う機能の場合は、「OSC 設定」を確認します。
4. 外部サービス側のログイン状態や権限も確認します。

## 連絡するときにあるとよい情報

1. 何をしようとしたか。
2. 何が起きたか。
3. 期待していた動き。
4. 問題が起きた時刻。
5. SleepVivo の画面名。
6. 設定した入眠予定時刻と起床予定時刻。
7. 可能であればスクリーンショット。

!!! note "報告は短くても大丈夫です"
    「この画面でボタンを押したら動かなかった」だけでも重要な情報です。
    Pre-Alpha 版では、細かい問題の報告が改善に直結します。ぜひご協力をお願いします。


## 困ったとき

解決できない場合は、Discord でハルジオン @fleabane_haru に連絡してください。
Research Program に参加しているかどうかに関係なく、通常のサポート連絡はできます。
