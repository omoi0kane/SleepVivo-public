# 自動化・連携[advanced]

「自動化・連携」は、SleepVivo の高度な設定を集約したページです。
VRChat、HMD、SteamVR、音、マイク、外部ツールなど、さまざまな機能をここから探せます。

基本的な入眠・起床だけなら、まず [睡眠サポート](sleep-induction.md) と [睡眠・起床の調整](sleep-wake-settings.md) を使ってください。

!!! warning "Advanced 設定です"
    ここにある機能は範囲が広く、環境によって動作が変わる場合があります。
    fork 元の OyasumiVR に由来する機能も多いため、一部の機能には不具合や SleepVivo 側の整理が残っている可能性があります。

!!! note "Research Program について"
    SleepVivo Research Program への参加は任意です。
    自動化・連携の利用に、Research Program への参加は必要ありません。

## 何をするページか

1. 画面・音・マイクに関する自動化を探します。
2. デバイス電源、HMD、GPU、シャットダウンなどの設定を探します。
3. VRChat に関する自動化を探します。
4. 外部連携や OSC、コマンド実行の設定を探します。
5. 検索や表示切替を使って、目的の設定へ移動します。

## 探し方

「自動化・連携」には、設定を探すための表示切替と検索があります。

1. 「有効なもの」を選ぶと、有効または設定済みの自動化だけを表示します。
2. 「状態 / タイミング別」を選ぶと、睡眠サポート開始時、睡眠確定時、起床確定時、予定・時刻ベース、イベントベース、常時参照のようなまとまりで探せます。
3. 「カテゴリ別」を選ぶと、画面・音・マイク、電源・性能、安全、VRChat 連携、外部連携のような分類で探せます。
4. 「難易度別」を選ぶと、調整向けか高度な設定かで探せます。
5. 検索欄の「設定を検索」に、設定名や説明に含まれる言葉を入力します。

## 分類の見方

各項目には、Core、Advanced、Legacy のようなバッジが表示される場合があります。

1. Core は、睡眠体験の中心に近い設定です。
2. Advanced は、さらに便利にしたい人向けの高度な設定です。
3. Legacy は、互換目的で残っている設定です。新しく使う場合は、通常の導線を優先してください。

## 画面・音・マイク

### 輝度と色温度

デバイス別の輝度上限、色温度制御、明るさ制御方式を設定します。
Valve Index や Bigscreen Beyond の輝度関連設定もここで確認します。

### 明るさの自動化

日の出、日の入り、HMD 接続などをきっかけにした明るさの自動化です。
睡眠サポートの基本的な明るさ調整とは別に、環境にあわせた動きを設定したい場合に確認します。

### マイクミュートの自動化

システムマイクの状態や、コントローラーバインドの挙動を設定します。
睡眠中や起床時のマイク状態を整理したい場合に確認します。

## 電源・性能

### デバイス電源

コントローラー、トラッカー、ベースステーションの電源制御を設定します。
睡眠中に不要なデバイスを止めたい場合や、SteamVR の起動・終了に合わせたい場合に確認します。

### HMD 固有機能

Bigscreen Beyond の fan / RGB など、対応 HMD 固有の機能を設定します。
対応していない HMD では効果がない場合があります。

### 解像度

睡眠確定時と起床確定時の SteamVR 解像度を設定します。
睡眠中の負荷を下げたい場合に確認します。

### フレームレート

アプリごとのフレーム制限を設定します。
睡眠中や起床時の負荷を調整したい場合に確認します。

### ガーディアン

SteamVR 境界のフェード距離自動化を設定します。
VR 空間での境界表示の見え方を調整したい場合に確認します。

### GPU

GPU 電力制限や MSI Afterburner 連携を設定します。
電力や発熱を抑えたい場合に使いますが、環境によって影響が大きいため注意してください。

### シャットダウン

PC や SteamVR を停止する条件を設定します。
睡眠中に自動で終了したい場合に確認します。

## 安全

### 悪夢検知

Pulsoid 連携時に、心拍数をもとにアラームや睡眠解除を行う設定です。
Pulsoid 連携が必要です。

## VRChat 連携

### ステータス

睡眠サポート開始時、睡眠確定時、起床確定時の VRChat ステータスを設定します。
ステータスメッセージの変更も含まれます。

### Invite と Req Invite

Invite / Req Invite の自動処理や、プリセット切り替えを設定します。
睡眠中の招待対応を調整したい場合に確認します。

### VRChat マイクミュート

OSC 経由で VRChat マイクの状態を自動制御します。
VRChat 内で OSC が有効である必要があります。

### アバター変更

睡眠サポート開始時、睡眠確定時、起床確定時のアバター切り替えを設定します。

### グループ変更

表示中の VRChat グループを自動で切り替えます。

### Join 通知

プレイヤーの入退室通知やサウンドを設定します。

### 睡眠アニメーション

姿勢 OSC と foot lock の自動化です。
アバター側の対応や OSC 設定が関係します。

## 外部連携

### 外部連携

VRChat、Pulsoid、Home Assistant、Discord、VRCX との接続を設定します。
外部サービスを使う機能は、先にここで接続状態を確認します。

### OSC 設定

OSC の送信先、カスタムターゲット、OSC サーバー受信を設定します。
VRChat 連携や OSC 自動化を使う場合に確認します。

### OSC 自動化

睡眠サポート開始時、睡眠確定時、起床確定時に任意の OSC スクリプトを送信します。
アバターや外部ツールに合わせた細かい制御をしたい場合に使います。

### コマンド実行

睡眠サポート開始時、睡眠確定時、起床確定時に Windows コマンドを実行します。
実行するコマンドの内容によっては PC の状態に影響するため、内容がわかる場合だけ使ってください。

## 使う前の確認

1. まず、やりたいことが [睡眠サポート](sleep-induction.md) や [睡眠・起床の調整](sleep-wake-settings.md) で足りるか確認します。
2. 外部サービスが必要な項目は、「外部連携」で接続状態を確認します。
3. OSC を使う項目は、「OSC 設定」も確認します。
4. ハードウェアや電源に関わる項目は、意図しない終了や電源操作が起きないように少しずつ試します。
5. 不安な項目は、変更前の状態をメモしてから試します。
