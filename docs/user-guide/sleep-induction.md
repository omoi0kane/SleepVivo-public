# 入眠誘導

入眠誘導は、寝る時刻に向けて明るさ、色温度、音量をやわらかく切り替え、眠りやすい状態へ整える機能です。
まずは左メニューの「睡眠サポート」から設定します。

## 基本設定

1. 左メニューの「睡眠サポート」を開きます。
2. 「今夜の設定」で「入眠サポート」を ON にします。
3. 「時刻の詳細」に表示される入眠時刻を確認します。
4. 時刻が未設定の場合は、「スケジュール設定を開く」から入眠予定時刻を入力します。
5. まずはおすすめプリセットの「標準」で試します。

## おすすめプリセット

「睡眠サポート」の「おすすめプリセット」では、入眠時の明るさと音量をまとめて選べます。

1. 「標準」は、初めて使うとき向けの基本設定です。
2. 「しっかり暗め」は、光が気になりやすい人向けです。
3. 「静かに寝る」は、音に敏感な人向けです。
4. 「明るめ」は、少し明るいほうが落ち着く人向けです。
5. 手動調整した睡眠サポート設定は「カスタム」へ保存されます。

## 画面と音を調整する

1. 「睡眠サポート」の「詳細設定」を押します。
2. 「睡眠・起床の調整」を開きます。
3. 「画面と音の調整」の「入眠誘導中」を確認します。
4. 「明るさ」を調整します。
5. 色温度制御を使っている場合は、「色温度」を調整します。
6. 「音量」を調整します。
7. 音量を変えたい再生デバイスがある場合は、「音量対象デバイス」で「デバイス選択」を押します。

!!! note "慣れるまでは細かい調整は不要です"
    入眠誘導の基本は、予定時刻とプリセットだけで使えます。
    数値を細かく変えるのは、明るさや音量が合わないと感じてからで大丈夫です。

## 時間を調整する

「睡眠・起床の調整」の「時間設定の調整」では、入眠誘導にかける時間を変えられます。

1. 「入眠までの遷移時間」を確認します。
2. 予定時刻までに、何分かけて入眠誘導を進めるかを入力します。
3. 「まだ寝ない」後の再誘導を使う場合は、何分後に再び入眠誘導するかを入力します。

## 詳細な誘導・睡眠設定

「詳細な誘導・睡眠設定」では、入眠誘導プロファイルの目標値を細かく設定できます。

1. 「明るさを変更する」を ON にすると、入眠誘導時の明るさを指定できます。
2. シンプル明度モードでは「簡易明度 (%)」を設定します。
3. 高度な明度モードでは「ソフトウェア明度 (%)」と「ハードウェア明度 (%)」を分けて設定します（Bigscreen Beyondのみ有効）
4. 色温度制御を有効にしている場合は、「色温度 (K)」を設定します。
5. 「音量を変更する」を ON にすると、入眠開始前の音量を 100% とした相対音量を指定できます。
6. 「手動実行時の遷移時間 (秒)」と「自動実行時の遷移時間 (秒)」を必要に応じて調整します。


## 困ったとき

入眠誘導が始まらない場合は、「入眠サポート」が ON か、入眠予定時刻が設定されているかを確認してください。
それでも解決しない場合は、設定画面の状態と予定時刻を添えて Discord でハルジオン @fleabane_haru に連絡してください。
