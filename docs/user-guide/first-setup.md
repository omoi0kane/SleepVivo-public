# 初回設定

SleepVivo は、基本的な睡眠サポートが最初から使える状態を目指しています。
初回は細かい設定を変更するより、まず睡眠・起床時刻を入れて動きを確認してください。

眠る前から起きるまでの全体像は、[SleepVivo の基本機能](basic-functions.md) を確認してください。

## 最初に設定すること

1. SleepVivo を起動します。
2. 左メニューの「睡眠サポート」を開きます。
3. 「入眠サポート」「寝落ち検知」「起床サポート」が必要に応じて ON になっているか確認します。
4. 「スケジュール設定を開く」を押すか、左メニューの「スケジュール」を開きます。
5. 「入眠予定時刻」と「起床予定時刻」を入力します。
6. まずはそのまま一晩使い、明るさや音量が合わない場合だけ「入眠サポート」から設定を調整してください。

## 固定時刻で使う場合

毎日ほぼ同じ時刻に寝起きする場合は、「スケジューラーを有効化」を OFF にしたまま使えます。
この場合は、固定の「入眠予定時刻」と「起床予定時刻」を設定します。

![固定時刻](../assets/img/schedule-fixed-time.png)


## 日によって予定が違う場合

日ごとに睡眠予定が変わる場合は、[スケジュール](scheduler.md)のスケジューラーが役に立ちます。
カレンダーで日ごとの予定を管理でき、「テンプレートから作成」や「まとめて設定」も使えます。

![スケジューラー](../assets/img/scheduler.png)


## 最初は触らなくてよい設定

1. 「詳細設定」は、明るさ、色温度、音量、遷移時間を細かく変えたいときに使います。
2. 「自動化・連携」は、VRChat、ハードウェア、外部連携を使い込みたい人向けです。
3. 「設定」の高度な項目は、トラブル対応や環境調整が必要なときだけ確認します。

!!! tip "迷ったら標準のまま"
    まずは「睡眠サポート」のおすすめプリセット「標準」を使うのがおすすめです。
    眠りにくい、明るすぎる、音が大きいなどがあれば、あとから調整します。


## 困ったとき

時刻を設定しても動きがわからない場合は、設定した入眠予定時刻、起床予定時刻、ON にした項目を添えて Discord でハルジオン @fleabane_haru に連絡してください。
