# 初回設定

SleepVivo は、基本的な睡眠サポートが最初から使える状態で設定されています。
初めて使う際は、まず睡眠・起床時刻を入れてみて、基本的な動作を確認してから、設定を変更することをおすすめします。

眠る前から起きるまでの全体像は、[SleepVivo の基本機能](basic-functions.md) を確認してください。

## 最低限設定が必要な項目

1. SleepVivo を起動します。
2. 左メニューの「スケジュール」を開きます。
3. 「入眠予定時刻」と「起床予定時刻」を入力します。
4. そのまま一晩使い、明るさや音量が合わないと感じたら「睡眠サポート」から設定を調整してください。

## 毎日決まった時刻で寝起きしたい場合

毎日ほぼ同じ時刻に寝起きする場合は、「スケジューラーを有効化」を OFF にします。
そして「入眠予定時刻」と「起床予定時刻」を設定します。

![固定時刻](../assets/img/schedule-fixed-time.png)

## 日によって予定が違う場合

日ごとに睡眠予定が変わる場合は、[スケジュール](scheduler.md)のスケジューラーを使います。
カレンダーで日ごとの予定を管理でき、「テンプレートから作成」や「まとめて設定」も使えます。
まずはカレンダーをクリックして、次の一晩の予定を入れてみましょう。

![スケジューラー](../assets/img/scheduler.png)

## 最初は触らなくてよい設定

1. 「詳細設定」は、明るさ、色温度、音量、遷移時間を細かく変えたいときに使います。
2. 「自動化・連携」は、VRChat、ハードウェア、外部連携を使い込みたい人向けです。
3. 「設定」の高度な項目は、トラブル対応や環境調整が必要なときだけ確認します。

!!! tip "迷ったら標準のまま"
    まずは「睡眠サポート」のおすすめプリセット「標準」を使うのがおすすめです。
    眠りにくい、明るすぎる、音が大きいなどがあれば、あとから調整も可能です。

!!! warning "自動化・連携は上級者向けの設定です"
    通常の使用では自動化・連携の設定は必要ありません。より便利にしたい時だけ使ってください。
    自動化・連携機能は範囲が広く、環境によって動作が変わる場合があります。
