# インストール

SleepVivo は現在 Pre-Alpha 版です。
現時点では、一般向けストア配布ではなく、インストーラーを使う配布形態です。
バージョンアップの際は、Windows＞プログラムのアンインストールまたは変更 から旧版を削除の上でインストールしてください。

!!! warning "配布方法は開発中です"
    このページは、現在のビルド成果物を受け取ってインストールする前提で説明します。
    正式なダウンロードページや自動更新の案内は、更新次第更新します。

## インストール前に確認すること

1. VR 環境と SteamVR が利用できる環境を準備してください。
2. 配布された SleepVivo のインストーラーをダウンロードしてください。
3. 以前の Pre-Alpha 版を入れている場合は、必要に応じて設定やログを控えた上で、設定情報を含めアンインストールしてください。

## インストール手順

1. 配布されたインストーラーを起動します。
2. 画面の案内に従ってインストールします。
3. インストール後、SleepVivo を起動します。
4. SteamVR を起動し、SleepVivo が正常に開けるか確認します。
5. 初めて使う場合は、[初回設定](first-setup.md)へ進みます。
![インストーラー](../assets/img/installer.png)

## 更新する場合

1. 新しいインストーラーを入手します。
2. SleepVivo を終了します。
3. 旧版をアンインストールしてください。設定データも削除してください。
4. 新しいインストーラーを実行します。

!!! note "Pre-Alpha 版の更新"
    更新方法や保存データの扱いは、ビルドごとに変わる可能性があります。
    配布時に個別の注意がある場合は、その案内を優先してください。

## アンインストール

1. Windows の「インストールされているアプリ」または「アプリと機能」を開きます。
2. SleepVivo を探します。
3. アンインストールを実行します。Pre-Alpha版では設定データもあわせて削除してください。

## 困ったとき

インストールに失敗した場合や、起動しても画面が出ない場合は、表示されたエラー、使っている Windows のバージョン、SteamVR の状態を添えて Discord でハルジオン @fleabane_haru に連絡してください。
Pre-Alpha 版のため、小さな問題でも報告をお願いします。
