# プライバシー

SleepVivo Research Program は任意参加の研究プログラムです。
参加しない場合でも、SleepVivo の通常機能は利用できます。

このページでは、Research Program におけるプライバシーの考え方について説明します。

## 基本方針

SleepVivo Research Program では、睡眠セッションの要約情報を収集し、分析に利用します。
氏名、メールアドレス、VRChat アカウント、音声、映像、生の HMD トラッキングデータのような直接的な情報は、研究データとして送信しません。

SleepVivoは皆さまのプライバシーに最大限配慮します。
ただし参加者の匿名ID、HMD モデル、睡眠傾向、通信時のサーバーログなどにより、法的・技術的に完全な匿名性が確保できない場合がありえます。
この場合、第三者への提供時は統計処理などにより個人識別がまったく不可能な形とするものとします。

## 収集される情報

Research Program が ON の場合、次のような要約情報が送信されます。

1. 睡眠開始日の粗い情報と開始時間帯
2. 睡眠時間
3. 睡眠品質、連続性、動き、姿勢、覚醒傾向、装着状態の要約指標
4. 一般化された HMD モデル情報
5. 同意バージョン、アプリバージョン、データ形式のバージョン

## 収集されない情報

Research Program の研究データとして、次の情報は送信しません。

1. 氏名、メールアドレス
2. VRChat アカウント名、VRChat ユーザー ID
3. 正確な位置情報
4. 音声、映像、スクリーンショット
5. 生の HMD トラッキングデータ、生の姿勢・IMU ストリーム（時系列データ）
6. HMD シリアル番号、PC 固有識別子
7. VR 内ワールド名、同席ユーザー名、チャット内容

## 匿名化・仮名化・集計

SleepVivo では、個人を直接識別できない形に加工した情報を扱います。
以下、必要に応じて「仮名化」「個人を特定できないよう加工」「集計済み」という表現を用います。

1. 仮名化は、直接の氏名やアカウント名の代わりに、ランダムな参加者 ID により特定個人の情報を扱う考え方です。
2. 集計は、複数のデータをまとめ、個別の参加者を特定しにくくする考え方です。
3. 匿名化という言葉を使う場合でも、完全な追跡不可能性を保証できない場合があります。

## 追跡不可能性と k 匿名性

追跡不可能性とは、データから特定の個人や行動履歴をたどれない状態を指します。
k 匿名性とは、同じような特徴を持つ人が少なくとも k 人以上いる状態にして、個人を目立ちにくくする考え方です。

Research Program では、正確な時刻や詳細すぎる HMD 情報を避け、粗い日付、時間帯、一般化された HMD モデルを使う方針です。
これは再識別の可能性を下げるための考え方です。

!!! warning "保証ではありません"
    これらはプライバシーリスクを下げるためのSleepVivoの方針です。
    完全な追跡不可能性や、特定の k 匿名性を保証するものではありません。

## 利用目的

送信された研究データは、次の目的で利用します。

1. SleepVivo の睡眠誘導・起床誘導・フィードバック機能の改善
2. HMD や利用環境ごとの VR 睡眠体験の傾向分析
3. VR 睡眠に関する統計的・研究的な分析
4. 不具合や品質低下の傾向把握
5. 個人を特定できない形に加工・集計した統計情報の共有、発表、共同研究

SleepVivoは医療診断、治療、健康状態の判定を目的としておらず、上記の用途には利用しません。

## 参加停止時の扱い

1. Research Program は、設定画面からいつでも OFF にできます。
2. OFF にすると、それ以後の研究データ送信は停止されます。
3. すでに送信されたデータは、個人を識別できない形に加工・集計された後は、個別削除できない場合があります。

## GDPR など個人情報保護法案への準拠について

現時点のバージョンでは、特定の個人情報保護法令への完全な準拠は想定しておりません。
正式なリリースに合わせて具体的な機能・対応・体制を整備します。


## 困ったとき

プライバシー面で不安がある場合は、Research Program を OFF のまま使ってください。
