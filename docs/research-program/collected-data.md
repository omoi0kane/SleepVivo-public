# 収集される情報

SleepVivo Research Program は任意参加です。
参加しない場合でも、SleepVivo の通常機能は利用できます。

このページでは、Research Program で送信される情報、送信されない情報、利用目的、参加停止時の扱いを説明します。

## 収集される情報

Research Program が ON の場合、送信される研究データは睡眠セッション単位の要約情報です。
送信される情報は、直接の氏名やアカウント名とは切り離し、参加者 ID などで仮名化・要約された形で扱います。
SleepVivoは皆さまのプライバシー保全に最大限の努力を払います。

1. 睡眠開始日のおおまかな情報
2. 睡眠開始時間帯
3. 睡眠時間、または睡眠推定時間
4. 睡眠品質に関する推定・要約指標
5. 睡眠の中断、安定性、連続性に関する指標
6. 睡眠中の身体・HMD の動きに関する指標
7. 睡眠中の姿勢傾向に関する指標
8. 起床・覚醒傾向に関する指標
9. HMD 装着状態の安定性に関する指標
10. HMD モデル情報
11. SleepVivoのユーザー同意/アプリ/データ形式のバージョン情報
12. VR Overlay アンケートに回答した場合の回答値

!!! note "送信データは要約情報です"
    研究データは、生の動きや音声ではなく、睡眠セッション単位で加工された要約情報です。
    （例：寝返りが何回打たれたか、睡眠途中で何回覚醒したか、など）
    皆さまの動作や音声が直接送られるわけではありません。

## VR Overlay アンケートで送信される情報

Research Program への参加は任意です。
参加していない場合、VR Overlay のアンケートは開けません。

参加中で回答対象の睡眠セッションがある場合、VR Overlay から睡眠後のアンケートに回答できます。
送信されるのは、質問の種類、1 から 5 の回答値、回答対象の睡眠セッションを結びつけるための情報です。

アンケートは、寝つきやすさ、睡眠への満足度、起きたときのすっきり感を確認するために使います。
回答しない場合でも、SleepVivo の通常機能は利用できます。

## 収集されない情報

Research Program の研究データとして、次の情報は送信しません。

1. 氏名
2. メールアドレス
3. VRChat アカウント名
4. VRChat ユーザー ID
5. 位置情報
6. 音声
7. 映像
8. スクリーンショット
9. 生の HMD トラッキングデータ
10. 生の姿勢・IMU ストリーム
11. HMD シリアル番号
12. PC 固有識別子
13. VR 内ワールド名
14. 同席ユーザー名
15. チャット内容

## 注意して扱う情報

仕様上、次の情報は取り扱いについて特に注意を払います。
ただし参加者 ID、HMD モデル、睡眠傾向、通信時のサーバーログなどにより、法的・技術的に完全な匿名性が確保できない場合がありえます。

1. 参加者 ID は、直接識別子と紐付けないランダム ID として扱います。
2. IP アドレスは、アプリの研究データ本体として保存しません。
3. 正確なタイムスタンプは避け、おおまかな日付や時間帯に丸めて記録します。
4. HMD モデルは、シリアルナンバーなどを含まず一般化して記録します。

## 利用目的

送信された研究データは、次の目的で利用します。

1. SleepVivo の睡眠誘導・起床誘導機能の改善
2. 睡眠フィードバック機能の改善
3. HMD や利用環境ごとの VR 睡眠体験の傾向分析
4. VR 睡眠に関する統計的・研究的な分析
5. 不具合、品質低下、装着不安定性の傾向把握
6. PoC、Alpha、Beta における仮説検証
7. 個人を特定できない形に加工・集計した統計情報の共有、発表、共同研究

SleepVivo Research Program は、医療診断、治療、健康状態の判定を目的としません。

## 参加停止時の扱い

1. 設定画面で Research Program を OFF にできます。
2. OFF にすると、それ以後の研究データ送信は停止されます。
3. 同意撤回後は、新しい研究データを送信しない方針です。
4. すでに送信されたデータは、個人を識別できない形に加工・集計された後は、個別削除できない場合があります。


## 困ったとき

送信される情報に不安がある場合は、Research Program を OFF のまま使ってください。
不明点は Discord でハルジオン @fleabane_haru に連絡してください。
