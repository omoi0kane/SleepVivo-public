# 同意と参加

SleepVivo Research Program への参加は任意です。
参加しない場合でも、SleepVivo の通常機能は利用できます。

Research Program を ON にするには、まず表示される同意画面をご覧ください。
内容をご確認の上、「同意して参加する」を押したのちに機能が ON になります。

## 参加する手順

1. 左メニューの「設定」を開きます。
2. 「一般設定」を開きます。
3. 「Research Program」を探します。
4. 「SleepVivo Research Program に参加する」を ON にします。
5. 「SleepVivo Research Program に参加する」という同意画面を読みます。
6. 送信される情報、送信されない情報、参加の停止を確認します。
7. 必要に応じて「詳細な説明を表示」を押します。
8. 参加する場合は「同意して参加する」を押します。
9. 参加しない場合は「参加しない」を押します。

## 同意画面で確認すること

同意画面では、次の内容を確認します。

1. Research Program は VR 睡眠体験の改善と睡眠傾向の研究を目的にしていること
2. 参加は任意であること
3. Research Program を OFF にしても通常機能は利用できること
4. 睡眠セッションの要約情報が送信されること
5. 氏名、メールアドレス、VRChat アカウント、音声、映像、生の HMD トラッキングデータは送信されないこと
6. いつでも設定画面から OFF にできること

詳しい文面はアプリ内からご確認ください

## 収集される情報

参加中は、睡眠セッションの要約情報が送信されます。

1. 睡眠開始日の粗い情報と開始時間帯
2. 睡眠時間
3. 睡眠品質、連続性、動き、姿勢、覚醒傾向、装着状態の要約指標
4. 一般化された HMD モデル情報
5. SleepVivoのユーザー同意/アプリ/データ形式のバージョン情報

## 収集されない情報

Research Program の研究データとして、次の情報は送信しません。

1. 氏名、メールアドレス
2. VRChat アカウント名、VRChat ユーザー ID
3. 正確な位置情報
4. 音声、映像、スクリーンショット
5. 生の HMD トラッキングデータ、生の姿勢・IMU ストリーム
6. HMD シリアル番号、PC 固有識別子
7. VR 内ワールド名、同席ユーザー名、チャット内容

## 利用目的

送信された情報は、次の目的で利用します。

1. SleepVivo の睡眠誘導・起床誘導・フィードバック機能の改善
2. HMD や利用環境ごとの傾向分析
3. VR 睡眠に関する統計的・研究的な分析
4. 不具合や品質低下の傾向把握
5. 個人を特定できない形に加工・集計した統計情報の共有、発表、共同研究

SleepVivoは医療診断、治療、健康状態の判定を目的としておらず、上記の用途には利用しません。

## 外部共有への同意

同意画面には、外部共有に関するチェックボックスがあります。
（匿名化・集計済みの研究結果を外部の研究・開発パートナーと共有することに同意します）

このチェックは Research Program 参加そのものとは別の同意です。
迷う場合は、内容を理解してから選んでください。

## 参加を停止する手順

1. 左メニューの「設定」を開きます。
2. 「一般設定」を開きます。
3. 「Research Program」を探します。
4. 「SleepVivo Research Program に参加する」を OFF にします。

OFF にすると、それ以後の研究データ送信は停止されます。
すでに送信されたデータは、個人を識別できない形に加工・集計された後は、個別削除できない場合があります。

## 困ったとき

同意画面の内容がわからない場合は、参加しないまま SleepVivo を使って問題ありません。
