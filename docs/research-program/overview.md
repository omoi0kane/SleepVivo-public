# SleepVivo Research Program 概要

SleepVivo Research Program は、VR 睡眠体験の改善と、VR 環境における睡眠傾向の研究を目的とした任意参加のプログラムです。
皆さんのデータが未来の新しい眠り方を作ります。ぜひご協力をお願いします。

!!! note "参加は任意です"
    Research Program を OFF にしても、入眠サポート、起床サポート、スケジュールなどの通常機能は利用できます。

## SleepVivo Research Programの目的

Research Programは、VR睡眠の研究を目的としたデータ収集、アンケート調査を行います。
Research Program に参加すると、SleepVivo は睡眠セッションの要約情報を送信します。
送信される情報は、VR 睡眠体験を改善するための分析に使用されます。

なお本プログラムは、医療診断、治療、健康状態の判定を目的とするものではありません。

## VR Overlay のアンケート

Research Program への参加は任意です。
参加していない場合、VR Overlay のアンケートは開けません。

参加中で回答対象の睡眠セッションがある場合、VR Overlay にクイズのボタンが表示されます。
アンケートでは、寝つきやすさ、睡眠への満足度、起きたときのすっきり感を 1 から 5 の数字で回答します。

回答は、VR 睡眠体験の改善と研究のために使います。
氏名、VRChat アカウント、音声、映像、スクリーンショットは送信されません。
参加を停止すると、それ以後のアンケート回答や研究データの送信は停止されます。

## 収集される情報

収集されるのは、睡眠セッション単位の要約情報です。
送信される情報は、直接の氏名やアカウント名とは切り離し、参加者 ID などで仮名化・要約された形で扱います。
SleepVivoは皆さまのプライバシー保全に最大限の努力を払います。

1. 睡眠開始日のおおまかな情報
2. 睡眠開始時間帯
3. 睡眠時間
4. 睡眠品質、連続性、動き、姿勢、覚醒傾向、装着状態の要約指標
5. 使用 HMD モデル情報
6. SleepVivoのユーザー同意/アプリ/データ形式のバージョン情報
7. VR Overlay アンケートに回答した場合の回答値

詳しくは [収集される情報](collected-data.md) を確認してください。

## 収集されない情報

Research Program では、次の情報は送信しません。

1. 氏名
2. メールアドレス
3. VRChat アカウント名、VRChat ユーザー ID
4. 位置情報
5. 音声、映像、スクリーンショット
6. 生の HMD トラッキングデータ、生の姿勢・IMU 情報
7. HMD シリアル番号、PC 固有識別子
8. VR 内ワールド名、同席ユーザー名、チャット内容

## 利用目的

送信された研究データは、次の目的で利用します。

1. SleepVivo の入眠サポート・起床サポート・フィードバック機能の改善
2. HMD や利用環境ごとの VR 睡眠体験の傾向分析
3. VR 睡眠に関する統計的・研究的な分析
4. 不具合、品質低下、装着不安定性の傾向把握
5. 個人を特定できない形に加工・集計した統計情報の共有、発表、共同研究

## 参加と停止

1. 「設定」から「一般設定」を開きます。
2. 「Research Program」を探します。
3. 「SleepVivo Research Program に参加する」を ON にします。
4. 同意画面をお読みください。
5. 同意の上で参加いただける場合は「同意して参加する」を押します。
6. 参加を中止する場合は、同じトグルを OFF にします。

OFF にすると、それ以後の研究データ送信は停止されます。
すでに送信されたデータは、個人を識別できない形に加工・集計された後は、個別削除できない場合があります。ご承知おきください。

詳しくは [同意と参加](consent.md) を確認してください。
