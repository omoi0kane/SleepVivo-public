# リリースノート

SleepVivo は現在 Pre-Alpha 版です。
画面や配布方法は今後変わる可能性があります。

## 既知の問題（known-issues）

[Github issue](https://github.com/omoi0kane/SleepVivo-public/issues)でも管理しています。
現在はPre-Alphaバージョンのため、全てのバグは管理していません。主要な問題は以下のとおりです。

1. 配布方法と更新方法は開発中です。
2. 自動更新は現在無効です。
3. 一部の Advanced / Legacy 設定は、環境によって動作が変わる場合があります。
4. スクリーンショットや説明文が、最新の画面と完全には一致しない場合があります。

## リリースノート

* 現在はインストーラー配布を前提にした公開ドキュメントを整備しています
* 睡眠サポート、スケジュール、設定、Research Program の説明を追加しています
* 既知の問題や更新方法は、配布時の案内を優先してください

### 0.2.3 Pre-Alpha (2026/5/2)

* Pre-Alphaリリースバージョン（限定配布）の更新版
* 上体起こし時に一定の条件で自動で起床モードに入る機能を追加
* 起床サポート時の明度遷移のタイミングを調整
* SteamVR終了時にSleepVivoが自動終了する設定をデフォルトに
* 睡眠ログの記録形式を拡張
* オーバーレイ表示404エラーに関するupstream由来のマイナーバグを修正(完全に治ったかは不明)


### 0.2.2 Pre-Alpha (2026/4/29)

* Pre-Alphaリリースバージョン（限定配布）の更新版
* 「自動化・連携」にHaritoraXでのフルトラ停止、電源OFF機能を追加
* プリセットの明度を低めに調整
* 「まだ寝ない」時の復帰トランジションを10秒へ短縮
* 起床サポート中に中断・復帰した場合、明度・色温度が復帰しないバグを修正
* 「今日の睡眠予定」が1日ずれる場合があるバグを修正

### 0.2.1 Pre-Alpha (2026/4/28)

* 初のPre-Alphaリリースバージョン（限定配布）
* 基本的な睡眠サイクル機能を実装
* VIをSleepVivoオリジナルに変更
