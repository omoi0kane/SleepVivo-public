# Public Shell

将来的に SleepVivo は一部のコードを透明性の観点で公開する可能性があります。
現在、shell 部分として公開されているコードはありません。

## 現在の状態

1. SleepVivo の公開ドキュメントは、この GitHub リポジトリで管理します。
2. shell 部分の公開コードは、現時点では存在しません。
3. 将来的に、公開対象、ライセンス、利用方法を確認したうえで案内します。

!!! note "現時点では予定です"
    Public Shell の公開は将来的な可能性です。
    実装済みまたは公開済みの機能として扱わないでください。

Public Shell について確認したい場合は、[サポート](../support/support.md) を確認してください。
