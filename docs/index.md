# SleepVivo

SleepVivo はVR睡眠支援ソフトウェアです。
HMDを活かした睡眠/起床サポートで、あなたのウェルネスをサポートします。

現在のドキュメントは Pre-Alpha 版を前提にしています。
画面や配布方法は今後変わる可能性があります。

![SleepVivo ホーム画面](assets/img/home.png)

!!! note "Pre-Alpha 版について"
    小さな違和感や不具合も改善の材料になります。
    気になる点があれば Discord でハルジオン @fleabane_haru に連絡してください。

## まずはこのページから

1. [インストール](user-guide/installation.md)で、現在の配布方法を確認します。

3. [SleepVivo の基本機能](user-guide/basic-functions.md)で、基本的な機能や睡眠サポートの流れを確認します。
4. [睡眠サポート](user-guide/sleep-support.md)で、入眠サポート、寝落ち検知、起床サポートを確認します。
5. 日によって予定が変わる場合は、[スケジュール](user-guide/scheduler.md)を確認します。

## ドキュメント構成

### ユーザーガイド

インストール、初回設定、睡眠サポート、スケジュール、設定をまとめています。
SleepVivo を使う人は、まずここから読んでください。

### Research Program

SleepVivo Research Program は任意参加のプログラムです。
VR 睡眠体験の改善と研究のため、同意した場合のみ睡眠セッションの要約情報を送信します。

参加しない場合でも、SleepVivo の通常機能は利用できます。
詳しくは [Research Program 概要](research-program/overview.md) を確認してください。

### サポート

よくある確認事項と連絡前に確認することをまとめています。
問題が起きた場合は [サポート](support/support.md) と [FAQ](support/faq.md) を確認してください。
