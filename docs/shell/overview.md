# 概要

将来的に SleepVivo は一部のコードを透明性の観点で公開する可能性があります。
現在、shell 部分として公開されているコードはありません。

## 現在の状態

1. SleepVivo の公開ドキュメントは、この Github リポジトリにて管理します。
2. shell 部分の公開コードは、現時点では存在しません。
3. 将来的に、fork 元の OyasumiVR 由来のコードや一部コードを公開する可能性があります。
4. 公開する場合は、公開対象、ライセンス、利用方法を確認したうえで案内します。

## 今後更新する内容

1. 公開対象が決まった場合の概要
2. 利用者が確認できるコードの範囲
3. ライセンスや利用条件
4. 問い合わせ先

!!! note "現時点では予定です"
    Public Shell の公開は将来的な可能性です。
    実装済みまたは公開済みの機能として扱わないでください。

## 困ったとき

Public Shell について確認したい場合は、Discord でハルジオン @fleabane_haru に連絡してください。
公開コードが存在するかどうかを確認したい場合も、現時点では「公開コードはありません」と理解してください。
