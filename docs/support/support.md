# サポート

!!! tip "不具合報告をお願いします"
    SleepVivo は現在 Pre-Alpha 版です。
    表示崩れ、ボタン反応が悪い、違和感、説明がわかりにくいなども改善に役立ちます。
    小さな点でもぜひ報告頂ければありがたいです。

## 連絡先

問題が起きた場合や設定で迷った場合は、Discord でハルジオン @fleabane_haru に連絡してください。

## サポートへ連絡する前に

1. (可能であれば) 問題が起きた画面のスクリーンショットを保存してください
2. 直前に変更した設定や行った動作があれば、内容をメモします
3. SleepVivoとSteamVR を再起動します
4. 問題が解消していないかを確認します
5. VR 機器が SteamVR から見えているか確認します

## 連絡するときにあるとよい情報

1. 何をしようとしたか
2. 何が起きたか
3. 期待していた動き
4. 問題が起きた時刻
5. SleepVivo の画面名
6. 設定した入眠予定時刻と起床予定時刻
7. 可能であれば問題が起きた画面と、ログ画面のスクリーンショット

!!! note "報告は短くても大丈夫です"
    「この画面でボタンを押したら動かなかった」だけでも重要な情報です。
    Pre-Alpha 版では、細かい問題の報告が改善に直結します。

## よくある確認先

1. 起動しない、設定が反映されない、通知や音量がおかしい場合は [FAQ](faq.md) を確認してください。
2. 睡眠サポートの基本は [睡眠サポート](../user-guide/sleep-support.md) を確認してください。
3. スケジュールの入力は [スケジュール](../user-guide/scheduler.md) を確認してください。
4. Research Program の送信内容は [収集される情報](../research-program/collected-data.md) を確認してください。
