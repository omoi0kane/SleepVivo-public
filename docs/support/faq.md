# FAQ

よくある質問の内容をまとめています。
解決できない場合は [サポート](support.md) へ相談してください。

## SleepVivoが起動しない

1. インストーラーが現在の配布元から受け取ったものか確認します
2. Windows の警告やエラー文を確認します
3. SteamVR を起動した状態でもう一度 SleepVivo を起動します
4. それでも起動しない場合は、表示されたエラー文を添えて連絡します

## 睡眠サポートが動かない

1. 左メニューの「睡眠サポート」を開きます
2. 「入眠サポート」「寝落ち検知」「起床サポート」が必要に応じて ON か確認します
3. 「時刻の詳細」に時刻が表示されているか確認します
4. 時刻が未設定の場合は、[スケジュール](../user-guide/scheduler.md)で入眠予定時刻と起床予定時刻を入力します
5. 日ごとの予定を使っている場合は、「スケジューラーを有効化」が ON か確認します

## スケジュールが反映されない

1. 「スケジュール」を開きます
2. 固定時刻で使っているか、スケジューラーで使っているか確認します
3. スケジューラーを使っている場合は、カレンダーの日付を選びます
4. 「就寝予定（前日）」と「起床予定」が入っているか確認します
5. 「まとめて設定」を使った場合は、対象期間と曜日を確認します

## 通知や音量がおかしい

1. 「設定」から「通知の設定」を開きます
2. 「通知先」を確認します
3. 「一般通知の音量」を調整し、「テスト」で確認します
4. 「睡眠・起床の調整」で「音量対象デバイス」が意図したデバイスになっているか確認します

## VRオーバーレイが開かない

1. SteamVR が起動しているか確認します
2. 「設定」>「一般設定」>「オーバーレイ」で「オーバーレイメニューを有効にする」が ON か確認します
3. コントローラーバインディングが設定されているか確認します
4. 「VRChatが実行中の場合のみメニューを開く」を ON にしている場合は、VRChat が起動しているか確認します
5. SteamVR ダッシュボードを開いたままにしていないか確認します

## 自動化・連携がうまくいかない

1. 「自動化・連携」の「有効なもの」で対象項目が表示されるか確認します
2. VRChat、Pulsoid、Home Assistant、Discord などの接続状態を確認します
3. OSC を使う機能の場合は、「OSC 設定」を確認します
4. 外部サービス側のログイン状態や権限も確認します

## プライバシーについて不安がある

SleepVivo は原則としてユーザーの情報を外部へ送信しません。
HMDから取得された睡眠などに関する情報は、ユーザーのPC内で処理されます。
ユーザー情報が外部送信されるのは、[SleepVivo Research Program](../research-program/overview.md) への参加に同意いただいた場合のみです。
例外は、Research Programへの参加を撤回した際の同意撤回メッセージのみです。

## SleepVivo Research Program とは？

VR 睡眠体験の改善と、VR 環境における睡眠傾向の研究を目的とした任意参加のプログラムです。
Research Program への参加は任意です。参加しない場合でも、SleepVivo の通常機能は利用できます。

送信される情報に不安がある場合は、Research Program を OFF のまま使ってください。
送信される情報と送信されない情報は [収集される情報](../research-program/collected-data.md) を確認してください。
